### **Python Lists – Key Learning Points (Bullet Points)**

* **Define & Manipulate Python Lists**

  * Create and initialize lists
  * Use essential list methods:

    * `append()`
    * `extend()`
    * `insert()`
    * `remove()`
    * `pop()`
    * `clear()`
    * `index()`
    * `count()`
  * Understand mutability and dynamic resizing of lists

* **Implement List Comprehensions**

  * Basic list comprehension syntax
  * With conditions (if/else)
  * Nested list comprehensions
  * Convert loops to list comprehension for cleaner code

* **Understand 1D Array Concepts**

  * Python list as a dynamic 1D array
  * Memory allocation & element access
  * Difference between list and array module basics

* **Perform Traversals & Indexing**

  * Iterate using `for` and `while`
  * Access elements using:

    * **Positive indexing** (`list[0]`, `list[1]`, …)
    * **Negative indexing** (`list[-1]`, `list[-2]`, …)
  * Perform slicing:

    * `list[start:end]`
    * `list[start:end:step]`

* **Solve Basic 1D Array Problems**

  * Find maximum & minimum elements
  * Calculate sum of elements
  * Linear search in list
  * Count occurrences
  * Check if an element exists

* **Practice Common List Operations**

  * Sorting lists using:

    * `sort()` (in-place)
    * `sorted()` (returns new list)
  * Reverse list:

    * `reverse()`
    * slicing (`list[::-1]`)
  * Copy lists using:

    * `copy()`
    * slicing (`list[:]`)
    * `list()` constructor
